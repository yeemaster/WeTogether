# WeTogether

>  基于React-Native的生活向APP。

## 项目描述
1. 基于React-Native框架开发，能同时运行在Android、iOS环境下。
2. 涵盖了图文、音乐、电影、咨询四大块.

## 整体思路
1. redux 管理数据流，react-navigation 路由。 项目文件的放置采取就近原则。

## Build
1. clone this repository
2. npm install
3. react-native run-android or react-native run-ios
